# 前端人脸活体检测（交互式）

## 1. 介绍

纯前端的人脸活体检测功能，可直接在浏览器中运行（打开摄像头功能如果不是https的情况下只能使用localhost）。支持4种类型的交互动作，分别是：点头，摇头，张嘴，眨眼。
